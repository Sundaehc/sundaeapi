# Sundae API开放平台后端

> 项目上线地址：http://43.143.69.192/

#### 项目模块介绍

1. sudaeapi-backend：项目主体，包括各项接口
2. sundae-interface：接口服务，用于实现接口调用功能
3. sundae-client-sdk：客户端sdk
4. sundaeApi-common：公共封装模块
5. sundaeapi-gateway：网关服务
